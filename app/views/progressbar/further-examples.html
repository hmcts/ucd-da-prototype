Notes:

This progress bar (vertical responsive) is an "extended" version of the HMCTS progress bar component

HMCTS design system component reference: HMCTS Progress bar

Steps to install:

1. Add the @hmcts module to your local node_modules folder if you do not already have this: Add @hmcts module. On compiling, it should appear in your local node_modules folder as @hmcts.

2. In file \app\assets\sass\application.scss add the progress bar component path: @import "components/progress-bar";

3. In folder \app\assets\sass\ create a folder called "components".

4. Download progressbar sass and tick png.

5. In folder \app\assets\sass\components\ add the scss file ( _progress-bar.scss ).

6. In folder \app\assets\images\ add the image ( icon-progress-tick.png ).

7. Copy the html code snippet ( opposite ) into your page, and customise as required.

8. Customise the _progress-bar.scss file as required.
