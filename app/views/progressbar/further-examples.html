Notes:

This progress bar (vertical responsive) is an "extended" version of the HMCTS progress bar component.

HMCTS design system component reference: HMCTS Progress bar

Steps to install:

1. Ensure you have added the hmcts node module to your local folder:@HMCTS design node module

2. In \app\assets\sass\ application.scss add the progress bar component path: @import "components/progress-bar";

3. In \app\assets\sass\ add a folder called "components"

4. Download progressbar sass and tick png

5. In \app\assets\sass\components\ add the scss file ( _progress-bar.scss )

6. In \app\assets\images\ add the image ( icon-progress-tick.png ).

7. The progress-bar sass provides a skeleton look and feel and can be customised as required.
